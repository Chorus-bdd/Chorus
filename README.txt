Chorus is a lightweight BDD test framework based upon the popular Cucumber BDD test framework for Ruby.

Key features:
 * Supports most commonly used keywords from Cucumber's Gherkin syntax
 * Provides additional support for testing highly complex Java distributed systems
 * Modular interpreter architecture that allows subprojects to extend/customise core features
 * Pure Java implementation
